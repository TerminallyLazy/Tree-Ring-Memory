## Reason

This directory is specific to the Agent Client Protocol ecosystem: ACP clients,
ACP-compatible agents, SDKs, registries, MCP integration for ACP agents, and ACP
learning resources.

Tree Ring Memory is local-first agent-memory CLI infrastructure. It does not
currently ship ACP support, an ACP client, or an ACP-compatible agent
implementation, so adding it now would misclassify the project.

## Next Action

Revisit only if Tree Ring Memory gains explicit ACP support or a companion ACP
integration.

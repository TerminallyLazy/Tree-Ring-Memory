## AI Engineering Pitch

```text
Tree Ring Memory is an open-source memory lifecycle framework for AI agents.
It is not a hosted memory service or vector DB. It focuses on how memory should
age: fresh context, compressed older rings, scars for important failures,
heartwood for durable truths, seeds for future hypotheses, and first-class
forgetting/redaction.

The current public runtime is local-first and Rust-native.

Website: https://terminallylazy.github.io/Tree-Ring-Memory/
Repo: https://github.com/TerminallyLazy/Tree-Ring-Memory
Launch feedback: https://github.com/TerminallyLazy/Tree-Ring-Memory/issues/26
```

## Local-First Pitch

```text
Tree Ring Memory is a local-first memory lifecycle layer for AI agents. Durable
memory lives locally by default in SQLite/FTS, with explicit write commands,
audit, forgetting, redaction, supersession, import/export, and no hidden
transcript recorder.

Website: https://terminallylazy.github.io/Tree-Ring-Memory/
```

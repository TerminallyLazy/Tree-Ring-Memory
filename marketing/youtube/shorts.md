Use these after the main YouTube demo is live. Each cut should end with the

## Short 1: Memory Should Age

Hook:

```text
Agent memory should not be a transcript dump.
```

Script:

```text
Most AI agent memory either disappears between runs or stores too much raw
context. Tree Ring Memory uses a lifecycle model instead: fresh work stays
detailed, older learning compresses, failures become scars, and durable truths
become heartwood.
```


## Short 2: Scars And Heartwood

Hook:

```text
Your agent should remember failures differently from facts.
```

Script:

```text
Tree Ring Memory separates evaluated outcomes. Promoted lessons can become
heartwood. Rejected experiments become scars, so future agents see the warning
instead of repeating the same failed path.
```

CTA:


## Short 3: Local-First Boundary

Hook:

```text
Agent memory does not need a hidden recorder.
```

Script:

```text
Tree Ring Memory is local-first by default. It does not scrape transcripts or
silently store terminal events. Durable writes are explicit: remember,
evidence, import, consolidation, maintenance, TUI action, or deliberate agent
action.
```
